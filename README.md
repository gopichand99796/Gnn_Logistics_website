# GNN Logistics – Interactive Logistics Landing Page

An immersive, story-driven logistics landing page built for **GNN Logistics Inc.** as part of the **E-Cell NIT Trichy Tech Domain Task**.

Unlike traditional corporate websites, this project transforms the logistics workflow into a visual storytelling experience. Visitors follow the journey of a shipment from port arrival through transportation networks and final delivery using smooth animations, modern UI design, and interactive scroll-based experiences.

## 📂 Project Setup

Clone the repository:

```bash
git clone <repository-url>
```

Navigate into the project:

```bash
cd gnn-logistics
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

Build for production:

```bash
npm run build
```

## 🎨 Design Philosophy

The objective of this project was to move beyond a conventional logistics website and create an engaging user experience that communicates the movement of cargo through visual storytelling.

Key principles:

* Storytelling over static content
* Motion-driven interactions
* Enterprise-grade aesthetics
* Clear user journey
* Responsive design
* Performance-focused implementation
